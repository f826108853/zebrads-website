# DESIGN.md — Zebrads 品牌官网

## 气质与意象

**意象锚点**：黎明的非洲草原上，一群斑马正在迁徙。黑白条纹随着奔跑节奏起伏，仿佛一张张穿过晨雾的数据流；远处橙色的旭日把斑马背脊勾出一道温热的轮廓光。条纹是秩序，也是身份——没有两只斑马的纹路相同，正如每一位被精准触达的用户。

**关键词**：野性、精准、动势、辨识度、黑白主导、暖光点睛。

**反审美**：拒绝"科技蓝 + 圆角卡片 + 紫蓝渐变"的 SaaS 套路；拒绝纯黑底炫光赛博风。Zebrads 是有体温的精准——黑白构成结构，琥珀色点亮转化。

## 视觉策略

- **条纹母题（Stripe Motif）**：斑马条纹作为贯穿全站的视觉系统。用法包括：Hero 背景的倾斜条纹叠层、Section 分隔的条纹切割线、数据卡片悬停时的条纹滑入、按钮的条纹悬停态、Footer 的密集横条带。
- **强对比块状排版**：大字号无衬线粗体标题，黑底白字与白底黑字的 Section 块交替出现，形成"条纹页面节奏"。
- **图形语言**：几何裁切、倾斜分割（约 -8°）、硬边阴影、单色线性图标（不用三色或拟物图标）。
- **图像方向**：不使用真实照片库，全部用 SVG/CSS 几何图形与渐变构造——斑马侧影剪影、数据网格、条纹波纹。

## 字体排版

- **中文主字体**：`Noto Sans SC`（思源黑体），权重用 400 / 600 / 800。
- **英文/数字展示字体**：`Space Grotesk`（用于品牌名 ZEBRADS、关键数据如 "120亿+"、"195+"），权重 500 / 700。
- **回退栈**：`PingFang SC`、`Microsoft YaHei`、`system-ui`。
- **排版节奏**：
  - Hero 主标题 `clamp(2.5rem, 6vw, 5.5rem)`，行高 0.95，字间距 -0.02em。
  - Section 标题 `clamp(2rem, 4vw, 3.5rem)`，800 字重。
  - 数字（如"120亿+"）使用 Space Grotesk + 倾斜衬条纹背景。
  - 正文 `1rem / 1.7`，次要文字降到 `0.875rem`。
  - 大量留白：Section 之间 `py-24 md:py-32`。

## 动效与交互

- **运动性格**：克制而有方向感——像斑马奔跑，不像气泡漂浮。所有动效要么是水平/对角滑动，要么是分阶段揭示。
- **缓动曲线**：统一使用 `cubic-bezier(0.22, 1, 0.36, 1)`（ease-out-expo 风格），出场快、落位稳。时长 500-700ms。
- **进入视口**：默认"从下方 24px 淡入上滑"，关键标题用"条纹遮罩从左向右揭开文本"。
- **悬停态**：
  - 按钮：白底变黑底（或反之），同时一条琥珀色细线从下方滑入。
  - 卡片：边框由 1px stone 变为 2px ink，整卡轻微上移 -4px，投下硬边阴影 `8px 8px 0 0 ink`。
  - Tab：激活态底部出现 3px 琥珀色条纹。
- **滚动**：Hero 背景的斜向条纹做缓慢 parallax，数字数据卡可做 counter up 动画。
- **加载/骨架**：用条纹 shimmer 代替常规灰色脉动。

## 布局与响应式

- 最大内容宽度 `max-w-7xl`（1280px），两侧 `px-6 md:px-10`。
- 移动端：导航折叠为抽屉，Hero 主标题降到 `2.5rem`，6 个优势卡片单列堆叠，行业卡片改为 2 列网格。
- 平板：3 列网格普遍变为 2 列。
- Section 节奏：黑底白字 → 白底黑字 → 米雾灰底 → 黑底，强迫滚动产生条纹节拍感。

## 组件规范

- **按钮**：直角或最大 4px 圆角（拒绝 16px+ 大圆角的 SaaS 感）；主 CTA = 琥珀色背景 + 黑色文字 + 右侧箭头图标；次级 = 黑底白字描边或反色。
- **卡片**：直角 8px 圆角上限，1px 硬描边，悬停加 4-8px 硬阴影偏移（不用模糊大阴影）。
- **图标**：`lucide-react` 单色线性，统一 1.75px 描边。
- **数据徽章**：琥珀色填充小标签，用于"NEW"、"热门"、分类 Tag。

## 设计禁忌

- 禁止使用紫蓝渐变、霓虹光晕、玻璃拟态（glassmorphism）。
- 禁止 16px+ 的"果冻圆角"按钮和卡片。
- 禁止彩色多色 emoji 风格图标；图标必须单色线性。
- 禁止使用真实人物照片（评价头像用首字母 + 条纹背景的几何头像替代）。
- 禁止三种以上的高饱和强调色同框出现；琥珀是唯一主强调，绿色只在数据场景。
- 禁止"渐变蒙版 + 粒子背景"组合的 Hero——用斑马条纹几何替代。
- 禁止柔和模糊阴影；阴影必须是硬边、有偏移的"贴纸式"。
